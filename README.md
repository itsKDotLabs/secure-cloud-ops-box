# Secure Cloud Ops Box

A hardened AWS EC2 Ubuntu instance designed for secure cloud operations using
IAM-based access, zero inbound network exposure, and centralized logging.

## Objectives
- Eliminate SSH access
- Enforce least-privilege IAM
- Harden Linux host
- Enable cloud-native logging
- Document security decisions

## Threat Model
- Credential theft
- Exposed management ports
- Privilege escalation
- Unauthorized access
